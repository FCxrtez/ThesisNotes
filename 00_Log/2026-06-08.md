# Testbench adder

En un principio solo vamos a comprobar a mano, ya que lo que quiero verificar es que la informacion entre etapas sea correcta. Una vez que todo el simulador este implementado podemos diseñar un script automatico que compare el output del modulo en sv, el modelo de python y la suma de python usando Numpy. Podemos establecer una cota de error, en base a la diferencia absoluta entre los resultados de los modulos y la libreria de python definimos si se paso el test o no.
