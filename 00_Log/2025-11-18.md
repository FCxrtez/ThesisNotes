# Ground, Round & Sticky bits

Se usan para la suma y multiplicacion de FP. si p es la precision de nuestra mantisa es decir

`x = [s, exp, man] = (-1)^s * 1.man * 2^(exp-bias)`

los tres bits se usan para soportar distintos tipos de modos de redondeo. Sea p la cantidad de bits de mantisa tenemos que luego de calcular la nueva mantisa (ya sea en mult o add):
`nueva_mantisa = pb'{...}+ G + R + S`

Donde: (round to nearest ties-to-even)

+ `G = 0` -> no need to examine R and S round down pues `< half`
+ `G = 1, R =0, S =0` -> exactly halfway
+ `G = 1, (R=1 or S = 1)` -> `> half` -> round up
