## tb Pattern

Para manejar todas las señales definimos la clase `VectorLoad`, que contiene funciones virtuales y el atributo `string filename`. Debajo de esta definimos `VectorLoadBus` y `VectorLoadSignal`, ambas son parametrizables en `WIDTH` pero la diferencia es que `VectorLoadBus` define:

```system verilog
    typedef struct packed {
        logic [WIDTH-1:0] op0;    
        logic [WIDTH-1:0] op1;
        logic [WIDTH-1:0] op2;
        logic [WIDTH-1:0] op3;
    } operands_bus;

    operands_bus assert_data [0:99];
```

Mientras que `VectorLoadSignal` define:

```system verilog
    logic [WIDTH-1:0] assert_data [0:99];
```

De esta manera manejamos todas las señales de assert que necesitaremos con dos clases.
