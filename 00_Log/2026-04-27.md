## Branch History Table

Predicts the outcome of conditional branches (T/NT). It acts as a cache storing historical behaviour of branch instructions, tipically indexed by the lower bits of the PC

## Branch Target Buffer

Small cache that stores the address of the destination of previously executed branch instructions. It is indexed by the lowest bits of the instrucion memory address.

## CVA6 Branch Prediction

Ariane cuenta con:

- **Branch History Table (BHT)**: Two bit saturation counter to predict branch outcomes.
- **Branch Taget Buffer (BTB)**: Stores target addresses for jumps.
- **Return Adress Stack(RAS)**: Predicts return addresses for function calls.

In the absence of a valid entry the core bases itself in backward branches taken and foward branches not taken. It performs preductions in the Instruction Fetch stage to feed back to the PC Generation stage. Branch mispredictions are detected in the Execute stage. In case of misprediction, the subsequent instructions in the pipeline must be flushed and the PC fetches the next intruction from the correct memory address to resume computation.

## gem5/MinorCPU branch prediction

The MinorCPU is a generic in-order execution model.


Sacado de [paper gem5](https://arxiv.org/pdf/2007.03152)
```
Currently, gem5
supports five different branch prediction techniques including the
well-known TAGE predictor as well as standard predictors such as
bi-mode, tournament, etc
```

**Fetch2 contains the branch prediction mechanism**.  The MinorCPU has no BHT or Return Address Stack (RAS). To instantiate a branch predictor, the `branchPredictor.py` object in python must be used, which internally uses the `2bit_local` predictor, the one most easily adaptable to the CVA6.

gem5 BPU behaviour for a branch instruction:

- Consult the BHT to obtain a prediciton
- If the prediciton is Taken, consult the BTB to obtain the branch target
- If there is a BTB or RAS miss, predict Not Taken. Otherwise, update the PC with the target from the entry

**If there is a BTB miss in gem5, the static prediction is Not Taken. The gem5 BHT is initialized to 0, this results in an initial Not Taken prediction.**
