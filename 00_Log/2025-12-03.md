# Firesim - Compilador\manaer\enerar bitstreams

Firesim vive dentro del ecosistema de software RTL simulators,

## Compilador

Golden gate es el compilador. Toma FIRRTL generado por Chipyard, emite verilog que toma vivado y **genera drivers c++ que interactuan con el en el Host??**. Vive en `/sim`.
Golden Gate is like Verilator but for FPGA-accelerated simulation. Generates c++ and verilog to simulte your design.
Otorga flexibilidad para hacer un trade-out entre tiempo/area. Podemos balancear tal que se genere un disenio de mas area para mejor tiempo.
Preparado para correr con FPGA Vitus?

## Manager

vive en `/deploy`. Top-level python donde se llama el comando fireSim y donde se almacena el sw para que se corra en la simulaciones a traves de definiciones de workloads.
