# Problema assert de save y carry

El operador de xor `^` es conmutativo y asociativo, pero la funcion de majority `MAJ` con la que se genera el carry si depende de como se ingresan los operandos. Pero lo que siempre debe cumplirse, lo que se podria agregar como un assert es que `save + carry = save + carry'` para un carry cualquiera generado por otro orden de operandos. De tdas formas el fix fue cambiar el orden de los operandos para que la logica sean lo mas similares posibles
