# Implementacion package testbench

Defini la clase  `TestBenchEnv` con todos la data de assert como atributos, ademas de un atributo de tipo `OutputHandler` que ahora contiene la logica de fetchear la hora actual y generar el archivo de output, asi como de loguear los mensajes. Dentro de `TestBenchEnv` defini dos metodos, new() que inicializa cada vector de test, aqui estan las referencias a los archivos que contienen la data del script de python. Ademas la funcion `load_all_vectors` que llama a la funcion virtual `load_data` de cada atributo heredada de `VectorLoad`. La idea para ejecutar los asserts es acceder a cada variable del Env, ya que si quisiera encapsularlo deberia definir una funcion para cada señal por limitaciones de system verilog.
